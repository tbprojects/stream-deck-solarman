How to find Grid ID & Auth Token
  1. Open globalhome.solarmanpv.com and log in.
  2. Open DevTools (F12) → Network tab.
  3. Reload the page and find a request to:
    /maintain-s/operating/system/{your-id}
  4. The number at the end of that URL is your Grid ID.
  5. In the request headers, copy the value after Bearer  from the Authorization header — that is your Auth Token.
